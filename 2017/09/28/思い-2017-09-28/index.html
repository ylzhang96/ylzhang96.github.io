思い 2017.09.28

PS: 夹杂着中日英三种语言的自我感慨。

大四的第一个月。

「いろいろあった」 という感じですね。

9月的前10天在赶大三夏季学期的项目，也是大学最后的一个团队项目。
常在想，大学强制的「团队」到底有多大的意义呢。如果不是志同道合的一群人，仅是为了完成一份作业，是很难凝聚在一起的。有的人有底线，即使不愿意的任务也能按时完成；有的人没有，甚至连最基本的个人展示都需要你帮他一字一句准备好；有的人自知理亏，前期没有任何声响，截止日时轻飘飘地说一声不会。他们占优势的，不过是知道你「比他们更在意这份成绩」而已，而你甚至没有任何方式去‘威胁’他们。
紧赶慢赶，昼夜不分，废寝忘食，最后总算在截止日前完成了项目。他们在截止时间前去吃了晚饭，你没吃午饭和晚饭，拜托父亲开车送你到学校，甚至在车上颠簸着看代码。他们说自己肚子疼去了医院，你忍着肚子疼在最后的58个小时内只睡了三个小时。委屈的感受在验收完后可能通通不见，但是那些个日日夜夜的折腾最终还是会在你的身体上留下印记。

9月中旬，生活总算回到了正轨。
早睡早起、准时吃饭、实习、建了个人主页、学学技术、读读书、准备保研、申请日签、商讨和朋友的聚会、换副新眼镜。一切都在渐渐变好，是一个相对非常轻松的大四。你开始希望整个大四都能如此有条不紊，开始期待整个未来都能如此波澜不惊，你开始写大四的规划，学日语的规划，学技术的规划。

9月下旬，进了医院。
腰酸、脊椎疼、腰酸、走几步路就头晕、喘不过气、心跳加速。等你意识到不能再拖了之前，你还在开保研的会，校医院的医生甚至建议你直接喊救护车去医院，生怕你昏倒在半路上。
没有剧烈运动的你的心跳跳到了127，血色素降到了常人的三分之一。这时你才意识到，「啊，我的嘴唇果然毫无血色啊。」
各种糟糕的生活习惯让你付出巨大的代价，所有的规划都成了空谈。医院里的你甚至闲不下心，想打开电脑学点什么，什么都好。你被紧急输了血，你的血管闭塞到根本抽不出血来。在你的身体里被源源不断输入血的时候，你甚至有种想哭的冲动。
医院里，你第一次全部看完了《今日の日はさようなら》。讽刺的是你也身处血液科的病床，讽刺的是你旁边床位的病人和耕太是一个病。
我们每天都在迎接明天，因为明天似乎总会到来。但当突然有一天我们只能用数字计算活着的概率时，当今天100%健康的人明天就会因为各种各样的原因死去时，我们该对身边人说些什么。
「死ぬことは、終わることじゃない。」
太致郁，嘈杂的病房里，你不得不匆忙擦了眼泪。

你曾希望在自己最好的年纪努力拼搏，赚很多钱养你的爱豆、为你的父母健康无忧、为你的孩子随心所欲。
你曾希望从明天开始好好执行规划、好好生活，但往往今天的变化会让你猝不及防。

现在的我想——
健康第一。再也没有什么比健康更重要的了。
及时行乐。花你认为合适的钱，追你认为值得追的星。
好好学习。保研后的你，该沉下心看看文章了，慢慢来，不要浮躁，不要急。

一切都是，慢慢来比较快。

写于2017.09.28
