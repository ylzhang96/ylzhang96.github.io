Tensorflow

发表于 2018-04-03 | 分类于 CS Learning

Introduction

  • 2015年11月，Google发布人工智能系统TensorFlow并宣布开源。
  • 可以在CPU和GPU上运行

数据流图（data flow graph）

  • 节点（nodes）和线（edges）的有向 图来描述 数学计算
  • 节点：数学操作、数据输入（feed in）、数据输出（push out）、读取/写入持久变量（persistent variable）
  • 线：结点之间的输入/输出关系，输运的 数据为size可动态调整的多维数据数组，即 张量（tensor）

使用

  • 使用图来表示计算任务
  • 在会话（Session）的上下文（context）中执行图
  • 使用tensor表示数据
  • 通过变量（Variable）维护状态
  • 使用feed和fetch为任意的操作赋值或获取数据

N2 文法

～あげく（に）

  • 動た形／名＋の　＋あげく（に）
  • 最后、后项多为消极的结果
  • 進路（しんろ）についてさんざん悩んだ あげく、大学院への進学（しんがく）をあきらめることにした。
  • 「あげくのはてに」同「あげく」、但后项结果比「あげく」更严重，且能用作接续词

~あまり（に）

  • 動辞書形／ナ形＋な／名＋の　＋あまり（に）
  • 由于程度过甚、导致某种结果、由于过于……而……
  • 彼女は人目（ひとめ）を気にする あまりに、嫌なことも嫌だと素直に言えない。
  • 大好きな大野智にやっと会えた私は、興奮（こうふん）のあまり、気を失（うしな）った。
  • 不接イ形，可将其变为名词形。如「嬉しい」⇒「嬉しさ」
  • 常与状态和感情的词一起使用，如「忙しさのあまり／驚きのあまり／嬉しさのあまり／喜びのあまり／悲しみのあまり／感激（かんげき）のあまり／心配のあまり」
